autocutsel

autocutsel synchronizes the two copy/paste buffers mainly used by X applications. It automates the use of xcutsel(1).
It scans each buffer regularly and synchonizes them.

I wrote this tool to have a unified copy/paste buffer between Windows and a VNC viewer displaying an X desktop.
